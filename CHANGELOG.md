# v0.1.1-beta.1

- Added advanced full-text translation with document analysis and editable
  AI-generated terminology.
- Added global and per-document glossary management.
- Added start, pause, resume, cancel, retry, progress, and task log controls.
- Added resilient multi-version translation caching and interrupted-task
  recovery.
- Added MinerU parsing and full-text translation actions to the PDF reader
  toolbar.
- Improved reader cache indicators and translation synchronization.

This is a prerelease intended for testing. The stable update channel remains
on `v0.1.0`.
